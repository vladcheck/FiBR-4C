# Магазин с товарами

Данный проект реализует одностраничный сайт со списком товаров. База данных уже содержит несколько созданных товаров. Пользователь может производить все CRUD действия, включая создание и удаление своих товаров.
Технологии: Express, axios, swagger, React, TS, scss

## Этапы работы

В течение практических работ 1-5 были разработаны:

- Бекенд на ExpressJS
- Фронтенд на React и TS
- Middleware между сервером и клиентом на базе Axios
- Документация эндпоинтов на Swagger (jsdoc)

## Запуск

```bash
npm install
# В одном терминале
npm run dev
# Во втором терминале
npm run server
```

## Окружение

Фронтенд работает на порту `3000`, бекенд (по умолчанию) на порту `3001`. Обращаться к ним можно по URL `localhost:3000` и `localhost:3001` соответственно.
Чтобы открыть Swagger, нужно запустить app.ts через `node ./src/api/app.ts` из корня проекта и перейти по URL `http://localhost:3001/api-docs/` (работает на базе бекенда, на порту `3001`).

## Эндпоинты

Краткое описание всех эндпоинтов:
| Метод | Эндпоинт | Действие |
| ----- | ------------- | ------------------- |
| GET | /api/products | получить все товары |
| GET | /api/products/:id | получить товар по id |
| POST | /api/products | создать новый товар |
| PATCH | /api/products/:id | обновить товар по id |
| DELETE | /api/products/:id | удалить товар по id |
